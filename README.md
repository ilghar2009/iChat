Real-Time Chat Room with Laravel & SSE
A lightweight, real-time chat application built with Laravel and Server-Sent Events (SSE). No WebSocket required.
Features:
- Real-time messaging using SSE
- Auto-cleanup after 3 hours - Lightweight - with Redis required -RTL support
  -Simple setup
  Requirements:
  -PHP 8.3+
  -Laravel 13.x
  -MySQL
